Getting Started Functions
write() writeInt() read() readInt() readFloat() randInt() search() length() repeat() quadcalc() area() areapoly() average() countint() countchar() intangle() sort()

EZC Librari is very easy to use.

Follow the steps below to use EZC Library:

1. Go to the download page.

2. Download the library.

3. Include the library in your header (#include "ezc.h").

4. And voila! The library is ready to use!

EZC Library has many functions that simplifies your program.

** There are example codes below every functions and you can see how to implement the code.

Select a functions from menu on the left hand side and see the usage and examples!

write() is a function you can use to print or show a statement to use. It's equivalent in c is printf. This function only allows you to print string data type.

EXAMPLE:::

Input: write("Hello World!")

Output: Hello World!

write() is a function you can use to print or show a statement to use. It's equivalent in c is printf. This function only allows you to print string data type.

EXAMPLE:::

Input: write("Hello World!")

Output: Hello World!

writeInt() is a function that can be used for printing integers. It is equivalent to the printf function in c, however it is simplified down to be easier to understand.

Example:::

Input: writeInt("32");

Output: 32

read() is a function which can be used for reading data inputted by the user. Its C equivalent is scanf.

EXAMPLE:::

Input: read(a);

Output: Takes in user input for a

readInt()is a function which can be used for reading data inputted by the user.

EXAMPLE:::

Input: read(int a);

Output: Takes in user input for a

randInt() allows the user to find a substring and its location within a specific string. This function does not exist in C as of now.

Example:::

Input: rand(50);

Output: Any random number between 0 - 50

search() is a function which allows a user to generate a random number when given a max value. In C, this requires the use of 10 or more lines, and this function simplifies it down so anyone can use it.

Example:::

Input: search("Hello World!","World");

Output: 6

length() allows the user to find the length of a specific string. This is a simplified version of its C equivalent, strlen.

Example:::

Input: length("Hi");

Output: 2

repeat() allows the user to create a repeat loop, which repeats a function a certain number of times. In C, this can only be achieved by creating a for loop with several variables.

Example:::

Input: repeat(function(){write("Hi!")},4);

Output: Hi!Hi!Hi!Hi!

quadcalc() allows you to get the two roots of a quadratic equation. To get the first root you need to specify it. quadcalc(a,b,c,d) => for a,b, and c, put the values of a, b, c from the equation you have. D specifies which root you want. For first root enter 1 and for second root enter 2.

Example:::

Input: quadcalc(3,4,-2,1);

Output: 0.387425

area() allows you to get the area of any polygon with only having the side. You just enter the side and returns the area. The second number you need to enter is the length of the side.

Example:::

Input: area(8, 6)

Output: 173.82

areapoly() allows you to get the area of a polygon by having the perimeter and the apothem. You give the apothem and the perimeter and it returns the area.

Example:::

Input: areapoly(8, 4.129);

Output: 16.516000

average() allows you to get the average of the integers in an array. You just give the name of the array and the number of integers in that array.

Example:::

Input: int array_name[] = {1,2,3}; average(array_name, 3);

Output: 2

countint() allows you to get the number of elements in an array. The array needs to have a data type of integer.

Example:::

Input: int array_name[] = {1,2,3,4,5}; countint(array_name);

Output: 5

sort() allows you to sort an array from least to greatest. You just need to enter lg for least to greatest or gl for greatest to least. Then you need to enter the name of the array (needs to be int), and then enter how many numbers that the array has.

Example:::

Input: int array_name[]={6,4,7,2}; sort("lg", array_name, 4);

Output: N/A // Does not have any output. But now array_name is sorted {2,4,6,7}
